# Student Tools Hub

A complete, modern, and production-ready suite of online tools designed specifically for students. 100% free, fast, and light.

## Features
- **Academic Tools**: GPA/CGPA calculators, Pomodoro timer, etc.
- **Math Tools**: Interest calculators, BMI, Age calculator, etc.
- **Writing Tools**: Word count, Case converter, etc.
- **Utility Tools**: Password generator, Unit converter, etc.
- **SEO Optimized**: Meta tags, Open Graph, and semantic HTML.
- **Mobile Responsive**: Works perfectly on all devices.
- **AdSense Ready**: Dedicated placeholders for monetization.

## Tech Stack
- HTML5
- CSS3 (Vanilla + Tailwind CSS)
- JavaScript (Vanilla)

## Project Structure
```text
student-tools-hub/
├── index.html           # Main Landing Page
├── pages/               # Legal and Contact pages
├── js/
│   ├── tools.js         # Tool logic
│   └── search.js        # Search and Filtering logic
├── css/
│   └── styles.css       # Custom styling
└── assets/              # Static assets
```

## How to use
Simply open `index.html` in any modern web browser.
